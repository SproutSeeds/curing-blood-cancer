# Curing Blood Cancer Starter Protocol

## Purpose

Create a public, reusable way to organize blood-cancer cure and vaccine
research questions without making patient-specific clinical recommendations.

## First Pass

1. Define the disease scope.
2. Identify accepted subtype names and adjacent diseases.
3. List public source anchors.
4. Separate standard care, trials, mechanisms, biomarkers, and hypotheses.
5. Label every claim as `fact`, `derived`, `hypothesis`, or `open-question`.
6. Publish only artifacts that pass the private-to-public workflow.

## Initial Outputs

- blood-cancer concept map
- multiple-myeloma treatment-class taxonomy
- clinical-trial query template
- blood-cancer cure/vaccine claim evidence schema

# Sources

Canonical public source registries live here.

Use these records to cite source anchors by stable local IDs instead of
duplicating URLs across artifacts.

## Current Registries

- [Source Registry v0](./source-registry-v0.md)

Every source record should state:

- source owner
- public URL
- source type
- blood-cancer scope
- what claims it can support
- what claims it cannot support
- access date

## Purpose

This proof plan defines how to test whether the MRD Resistance Geometry is a
useful moving research structure.

It does not try to prove a cure, a treatment, a prognosis, a diagnostic
interpretation, or clinical validity. It tests whether the geometry can keep
source-backed observations separate, move under new evidence, survive
contradiction pressure, and fail closed when a requested output would become
clinical or patient-specific.

## What Proof Means Here

For v0, proof means the system passes five research-structure tests. Passing
does not validate any clinical claim.

| Test | Pass Condition | Failure Action |
| --- | --- | --- |
| `structural-separability-test-v0` | Measurement trajectory, residual biological state, clone architecture, and clone-state coupling remain separate when new sources are added. | Split the overloaded node or edge; block claim upgrade. |
| `productive-movement-test-v0` | A new public extraction can move an under-covered node to `covered-for-v0-navigation` while preserving source context and nonclinical boundaries. | Keep the node under-covered and add a source-specific gap. |
| `contradiction-response-test-v0` | Contradiction mining can weaken, split, or block an edge without breaking the whole geometry. | Downgrade the affected edge and route a follow-up extraction. |
| `state-object-fit-test-v0` | The residual-state object can represent source extraction context without losing uncertainty, missingness, or blocked outputs. | Add required fields or tighten the object boundary. |
| `safety-invariance-test-v0` | No geometry movement produces diagnosis, prognosis, treatment, monitoring, trial, ranking, patient matching, clinical validation, or cure output. | Block the output and revise the boundary language. |

## Frontier Progress

The frontier step here is structural, not clinical.

The system turns literature into a moving geometry that can be attacked and
updated without becoming a recommendation engine. That matters because MRD,
residual malignant cell state, clone architecture, therapy pressure, and
microenvironment context can all point at the same hard biological problem
while still needing different evidence rules.

The progress so far is that uncertainty is no longer just prose. It has places
to go:

- under-covered branches stay under-covered instead of becoming weak claims
- contradiction questions become tasks
- measurement movement stays separate from biology movement
- clone-state coupling stays an edge instead of a ranking
- pathway signals stay descriptive instead of actionable
- missingness is preserved in the residual-state object
- unsafe outputs are explicitly blocked

This gives us a research instrument that can become more precise as sources
arrive. The useful question becomes: which node moved, why did it move, which
source supported the move, which contradiction would break it, and which output
remains blocked?

## What This Does Not Prove

- It does not prove that multiple myeloma has been cured.
- It does not prove a treatment, target, biomarker, assay, supplement, pathway,
  or trial strategy.
- It does not estimate patient relapse risk.
- It does not interpret any patient MRD result.
- It does not rank mechanisms by clinical importance.
- It does not replace clinical review or expert validation.

# Multiple Myeloma Disease Program

Stewarded by [frg.earth](https://frg.earth/).

Multiple myeloma is the first disease program inside Curing Blood Cancer.

The goal is not to claim a cure. The goal is to build public tools that make
the cure-oriented questions easier to define, inspect, score, and improve from
transparent sources.

## First Cure Wedge

[Durable MRD-negative remission and relapse prevention](cure-wedge-durable-mrd-negative-remission-v0.md)

This wedge asks where current and emerging myeloma strategies produce deep
responses, why those responses stop being durable for some people, and which
research directions deserve structured public tracking.

## Program Boundaries

- Not medical advice.
- Not a treatment recommendation.
- Not a trial recommendation.
- Not a claim that multiple myeloma has been cured.
- Public artifacts must preserve source IDs, uncertainty, and disease-state
  scope.

## Working Directories

- `evidence-claims/`: structured claims that carry provenance and uncertainty
- `mechanisms/`: mechanism maps and relapse-biology notes
- `trial-maps/`: trial-query outputs and derived public trial landscapes
- `opportunity-map-v0.json`: first scored map of research/tooling opportunities

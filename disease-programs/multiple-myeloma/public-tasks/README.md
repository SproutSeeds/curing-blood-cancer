registers. Tasks are written so contributors can open safe GitHub issues,
prepare source extractions, or submit review-oriented pull requests without
private repo context.

## Boundary

- Research tasks only.
- Not medical advice.
- Not diagnostic guidance.
- Not a treatment recommendation.
- Not a trial recommendation.
- Not evidence ranking for patient decisions.
- Tasks must preserve source IDs, gap IDs, claim IDs, measurement-term IDs, and
  safety boundaries when available.

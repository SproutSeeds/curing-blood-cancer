## Acceptance Pattern

Each task should produce one source-specific contradiction note or extraction
that tests whether an edge should remain unified, split, weakened, or blocked.

The work must preserve:

- source IDs
- assay method
- specimen and sampling context
- threshold or sensitivity context when available
- cohort and therapy-exposure context
- source-defined response labels
- nonclinical boundaries

## What This Queue Does Not Do

- It does not open issues or request external expert review.
- It does not complete contradiction mining.
- It does not validate any claim for clinical use.
- It does not rank mechanisms, pathways, therapies, assays, sources, or tasks.
- It does not create treatment, monitoring, diagnostic, trial, prognosis, or
  patient-specific guidance.

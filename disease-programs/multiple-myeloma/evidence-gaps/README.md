# Multiple Myeloma Evidence Gaps

Stewarded by [frg.earth](https://frg.earth/).

This directory turns public claim sets into concrete work queues. Gap registers
say what the current artifacts do not yet establish, what public work would
improve confidence, and what readers should not infer from the current record.

## Current Gap Registers

- [BCMA Antigen Escape Evidence Gap Register v0](bcma-antigen-escape-evidence-gap-register-v0.md)

## Boundary

- Research artifacts only.
- Not medical advice.
- Not diagnostic guidance.
- Not a treatment recommendation.
- Not a trial recommendation.
- Not evidence ranking for patient decisions.
- Every gap should point back to source IDs, claim IDs, and measurement terms
  where possible.

# Multiple Myeloma Case Intake

Stewarded by [frg.earth](https://frg.earth/).

This directory holds public-safe caregiver intake specifications for the
multiple myeloma program.

These artifacts do not collect real case data. They define the public product
surface, synthetic examples, private handoff contracts, and safety boundaries
needed before a private, consented, access-controlled intake system exists.

## Boundary

- No patient-identifying data.
- No real records, notes, dates tied to a person, images, portal exports, or
  re-identification keys.
- No diagnosis, prognosis, treatment, trial, monitoring, urgency, or
  expanded-access advice.
- Public examples must be synthetic and visibly artificial.
- Real intake belongs only in a private, consented, access-controlled system
  with clinician, privacy, legal, regulatory, and safety review.

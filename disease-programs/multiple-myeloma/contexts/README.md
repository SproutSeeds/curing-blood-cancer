# Multiple Myeloma Context Modifier Maps

Public-safe context artifacts for high-risk, extramedullary, organ, frailty,
performance-status, immune-status, infection-risk, disease-burden,
disease-site, and prior-exposure language belong here.

These artifacts are research-use-only. They are not prognosis guidance,
eligibility guidance, treatment advice, trial advice, monitoring guidance, or
option ranking.

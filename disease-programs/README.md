# Disease Programs

Stewarded by [frg.earth](https://frg.earth/).

Disease programs are focused lanes inside the broader Curing Blood Cancer
initiative.

Each program should define:

- disease and subtype scope
- cure-oriented outcome questions
- public source anchors
- reusable evidence claims
- trial and literature mapping protocols
- safety boundaries for public use

The umbrella repo remains blood-cancer-wide. Disease programs keep the work
scientifically narrow enough to build useful artifacts.
